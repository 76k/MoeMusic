## Shortcut keys

| keys | function     |
| :------------- | :------------- |
| `ctrl+m` & `⌘+m`       | Global menu  |
| `ctrl+a` & `⌘+a`       | Playlist menu |
| `ctrl+p` & `⌘+p`       | Global player |
| `ctrl+f`       | search |
| `esc` | exit all Two level menu |

## Build & Dev
In testing(`don't it install ;(`)...
```bash
  cd project
  npm install
  npm run server
  npm server
  npm start

  # build
  npm build-all
```
